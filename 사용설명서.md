## 대시보드 접속

**https://dashboard-delta-two-72.vercel.app**

- 어떤 기기, 어떤 브라우저에서도 접속 가능
- 5초마다 자동 갱신 (탭이 활성화된 상태에서)

---

## 수업 전 준비

### 1. 오늘 퀴즈 등록 (`할일배정` 탭)

`확인학습 마스터` 스프레드시트 → `할일배정` 탭

| A (학교) | B (학년) | C (퀴즈제목) | D (폼링크) |
|----------|----------|--------------|------------|
| 유성고   | 1        | 문법-시제가정법 | https://forms.gle/xxx |

- **C열(퀴즈제목)**: 구글폼 제목과 **글자 하나까지 동일**해야 함
- **D열(폼링크)**: 학생에게 공유할 폼 링크
- 어제 내용 그대로 두면 어제 퀴즈가 오늘도 배정됨 → 매일 교체 필요

### 2. 학생 상태 초기화 (`학생명단` 탭)

- D열 상태를 `출석`으로 리셋
- 결석 예정 학생은 미리 `결석`으로 변경

### 3. 캐시 초기화 (선택)

`📊 대시보드` 메뉴 → `캐시 초기화`
(안 해도 5분 후 자동 만료)

## 수업 중 모니터링

### 대시보드 보는 법

```
┌─────────────────────────────────────┐
│ 완료 12  진행중 3  미시작 8  결석 2  │  ← 통계 바
│ 평균 78점  70%미만 4명               │
└─────────────────────────────────────┘

김OO  유성고1  ██████░░  2/3  85%     ← 완료 학생
이OO  대전고2  ░░░░░░░░  0/2  —       ← 미시작 학생
```

- **초록**: 90% 이상
- **파랑**: 70~89%
- **주황**: 50~69%
- **빨강**: 50% 미만
- **⏳**: 점수 집계 중 (5분 내 자동 보정)

### 미완료 학생에게 링크 보내기

1. 필터 → `미완료만` 선택
2. `미완료 링크 복사` 버튼 클릭
3. 복사된 내용을 카카오톡/문자로 전송

### 학생 상태 변경

학생 카드 클릭 → `결석 / 귀가 / 지각 / 추가문제` 버튼

## 자주 쓰는 기능

### 점수 직접 수정

학생 카드 → 퀴즈 항목 클릭 → `점수 수정` → 새 점수 입력

### 수동 완료 처리

학생이 폼 없이 제출한 경우:
학생 카드 → 퀴즈 항목 → `수동 완료` 버튼

### 학생 검색

상단 검색창에 이름 또는 학교명 입력

## 문제 상황 대처

| 증상 | 원인 | 해결 |
|------|------|------|
| 폼 제출했는데 대시보드에 안 나옴 | 트리거 미등록 | `📊 대시보드` → `트리거 등록` |
| 이름이 "미매칭"으로 표시 | 학생이 이름 오타 입력 | Summary 탭에서 B열 직접 수정 |
| ⏳ 5분 넘게 지속 | 채점 서버 오류 | Summary H열 `점수대기` 삭제 후 D~F열 직접 입력 |
| 연결 오류 | GAS 배포 만료 | Apps Script → 배포 관리 → 새 버전 재배포 |
| 어제 데이터가 그대로 | 할일배정 미교체 | `할일배정` 탭 내용 교체 |

| 탭 이름 | 역할 | 직접 수정 여부 |

| `주간과제` | 주간 과제 목록 | ✅ 수정 가능 |
| `문법-시제가정법` 등 | 폼 응답 원본 | ❌ 수정 금지 |
