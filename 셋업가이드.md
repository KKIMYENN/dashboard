# 양영학원 실시간 확인학습 대시보드 — 셋업 가이드

## 전체 구조

```
학생 폰/PC → 구글폼 제출 → 마스터 스프레드시트에 자동 기록
                                    ↓
                        Apps Script → Summary 탭 집계 + 캐시 갱신
                                    ↓
                대시보드 (Vercel 배포, 5초마다 갱신) → 선생님 모니터링
```

- **데이터 원본**: 구글 스프레드시트 (모든 기기가 동일한 데이터 참조)
- **대시보드 URL**: https://dashboard-delta-two-72.vercel.app
- **백엔드 URL**: `https://script.google.com/macros/s/AKfycbz4jSVmCXsyDrL6m70BDMCCVwzS9dRT7BLIyg_emfLGUm99MK0hR6gSgXMftsrjt5PW2A/exec`

---

## 1단계: 마스터 스프레드시트 생성

1. **새 Google 스프레드시트** 생성
   - 현재 운영 중인 시트: `확인학습 마스터`

2. **Apps Script 열기**
   - 메뉴 → `확장 프로그램` → `Apps Script`
   - 기존 코드 전부 삭제 후, `Code.gs` 파일 내용 전체를 붙여넣기
   - 저장 (Ctrl+S)

3. **초기 설정 실행**
   - 스프레드시트로 돌아가면 메뉴에 `📊 대시보드` 메뉴가 보임
   - `📊 대시보드` → `시트 초기 설정` 클릭
   - 권한 승인 팝업 → 승인
   - `학생명단`, `할일배정`, `Summary`, `기간관리`, `주간과제` 탭이 자동 생성됨

---

## 2단계: 학생 명단 입력

`학생명단` 탭에 학생 정보 입력:

| A (이름) | B (학교) | C (학년) | D (상태) |
|----------|----------|----------|----------|
| 김OO     | 유성고   | 1        | 출석     |
| 박OO     | 대전고   | 2        | 출석     |
| 이OO     | 충남고   | 1        | 출석     |

- **D열(상태)**: 기본값 `출석`. 드롭다운으로 `결석 / 귀가 / 지각 / 추가문제` 선택 가능
  - 대시보드에서도 버튼으로 변경 가능

---

## 3단계: 구글폼 만들기

### 폼 규칙 (반드시 지켜야 할 것!)

1. **퀴즈 모드 ON**
   - 폼 설정 → `퀴즈로 만들기` 활성화
   - 자동 채점 (정답 설정 필수)

2. **첫 번째 문항: "이름"**
   - 단답형(주관식)
   - 필수 응답 ON
   - 점수 0점 (채점 대상 아님)
   - 학생이 직접 이름 입력

3. **나머지 문항**: 객관식/단답형 자유롭게

### 폼을 마스터 스프레드시트에 연결

1. 폼 편집 화면 → `응답` 탭
2. 스프레드시트 아이콘 클릭 → `기존 스프레드시트 선택`
3. **1단계에서 만든 마스터 스프레드시트** 선택
4. 각 폼마다 별도 탭이 자동 생성됨 (예: `문법-시제가정법`)

---

## 4단계: 할일배정 입력

`할일배정` 탭에 오늘 풀어야 할 퀴즈 목록 입력:

| A (학교) | B (학년) | C (퀴즈제목) | D (폼링크) | E (폼ID) | F (대상학생) |
|----------|----------|--------------|------------|----------|--------------|
| 유성고   | 1        | 문법-시제가정법 | https://forms.gle/xxx | | |
| 유성고   | 1        | 독해-모의3차   | https://forms.gle/yyy | | |
| (비워도 됨) | | 단어-3주차 | https://forms.gle/zzz | | 김OO,박OO |

- **C열(퀴즈제목)**: 구글폼의 제목과 **정확히 일치**해야 합니다
- **D열(폼링크)**: 학생에게 보낼 구글폼 링크 (점수 자동 보정에도 사용)
- **F열(대상학생)**: 특정 학생에게만 배정할 경우 쉼표로 구분해서 입력. 비워두면 해당 학교+학년 전원

---

## 5단계: 트리거 등록

1. 스프레드시트 → `📊 대시보드` → `트리거 등록`
2. 권한 승인
3. 이후 자동으로 등록되는 트리거:
   - **onFormSubmitHandler**: 폼 제출 시 즉시 Summary에 기록
   - **fixMissingScores**: 5분마다 실행, 점수 누락 건 자동 보정

> **새 폼을 추가한 후에도** 트리거를 다시 등록할 필요 없습니다.
> 스프레드시트 단위 트리거이므로 연결된 모든 폼에 자동 적용됩니다.

---

## 6단계: 웹 앱 배포

1. Apps Script 에디터로 이동
2. 우측 상단 `배포` → `새 배포`
3. 설정:
   - 유형: `웹 앱`
   - 설명: `확인학습 대시보드`
   - 실행 주체: `나`
   - 액세스: `모든 사용자` (구글 로그인 불필요)
4. `배포` 클릭
5. **웹 앱 URL** 복사 (`https://script.google.com/macros/s/.../exec`)

> 코드 수정 후에는 반드시 `배포 관리` → ✏️ 편집 → 버전 `새 버전`으로 재배포해야 적용됩니다.

---

## 7단계: 대시보드 접속

**https://dashboard-delta-two-72.vercel.app** 접속

- 처음 접속 시 GAS 웹 앱 URL 입력 창이 뜰 수 있음
- 6단계에서 복사한 URL 붙여넣기 → 저장
- URL은 브라우저 localStorage에 저장되므로 다음부터는 자동 접속

> 여러 기기에서 동시 접속 가능. 모두 같은 스프레드시트 데이터를 참조합니다.

## 일상 운영

### 새 날 시작할 때

1. `할일배정` 탭의 내용을 오늘 퀴즈로 교체
2. `학생명단` 탭의 D열 상태를 `출석`으로 리셋
3. `📊 대시보드` → `캐시 초기화` (선택사항, 5분 후 자동 만료)

### 새 구글폼 추가할 때

1. 위 3단계대로 폼 생성 + 마스터 시트에 연결
2. `할일배정` 탭에 새 퀴즈 행 추가
3. 폼 제목과 할일배정 C열이 일치하는지 확인

### 학생 추가/삭제

- `학생명단` 탭에서 직접 행 추가/삭제

| 기능 | 설명 |
|------|------|

## 트러블슈팅

### "연결 오류" 표시

- GAS 웹 앱 URL이 올바른지 확인
- `배포 관리` → 최신 버전이 활성인지 확인

### 폼 제출했는데 대시보드에 안 나옴

1. Summary 탭에 행이 추가되었는지 확인
2. 추가 안 됨 → `📊 대시보드` → `트리거 등록` 다시 실행
3. 추가됨 but 대시보드에 안 보임 → 이름이 `학생명단`과 정확히 일치하는지 확인
4. 이름 불일치 → 대시보드 상단에 "미매칭" 경고로 표시됨

### 점수가 0점 / ⏳ 표시

- 구글 채점 서버가 늦게 처리한 경우. **5분 내 자동 보정**됨
- 5분 후에도 안 되면 Summary 탭에서 H열 `점수대기` 확인 후 수동 수정

### 할일배정과 폼 제목 불일치

- `할일배정` C열의 퀴즈 제목과 구글폼 제목이 **글자 하나까지** 일치해야 합니다
- 공백, 하이픈(-) 등 주의

### 캐시 문제

- `📊 대시보드` → `캐시 초기화` 후 다시 확인
- 캐시는 5분(300초) 후 자동 만료

### Apps Script 코드 수정 후

- 코드 수정 → `배포` → `배포 관리` → ✏️ → 버전 `새 버전`으로 → 배포
- **반드시 새 버전으로 배포**해야 변경사항 적용됨
