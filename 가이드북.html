🚀 실시간 확인학습 대시보드 가이드북

선생님과 학생을 위한 실시간 퀴즈 및 과제 관리 시스템 완벽 설정 가이드

📂 전체 시스템 구조

대시보드는 크게 세 가지 요소로 맞물려 돌아갑니다.

  1. 학생 환경: 스마트폰이나 PC로 구글 폼에 접속해 퀴즈를 제출합니다.
  2. 마스터 스프레드시트 (서버): 학생들의 응답과 점수가 자동으로 집계되며 백그라운드 코드가 처리합니다.
  3. 대시보드 HTML (프론트엔드): 선생님이 띄워놓는 화면으로, 5초마다 마스터 시트와 통신하여 결과를 갱신합니다.

✨ 단계별 초기 셋업 가이드

1단계. 마스터 스프레드시트 생성

  • 구글 드라이브에서 새 Google 스프레드시트를 만듭니다. (이름: ★확인학습 마스터 등)
  • 상단 메뉴에서 확장 프로그램 ➔ Apps Script를 엽니다.
  • 미리 제공된 Code.gs 소스코드를 전체 복사하여 붙여넣고 저장(Ctrl+S)합니다.
  • 스프레드시트로 돌아오면 상단에 새로 생긴 📊 대시보드 ➔ 시트 초기 설정 메뉴를 누릅니다. (권한 승인 필요)
  • 학생명단, 할일배정, Summary 시트가 자동으로 생성됩니다.

2단계. 구글 폼 만들기 및 연결

⚠️ 가장 중요합니다! 구글 폼 작성 규칙
1. 폼 설정에서 반드시 '퀴즈로 만들기'를 켜주세요. (주관식 단답형/객관식 정답 세팅 필수)
2. 첫 번째 문항은 무조건 '이름' (단답형, 필수 응답, 0점)이어야 합니다. 학생이 타자로 직접 입력해야 매칭됩니다.
  • 구글 폼 편집기 화면 중앙 '응답' 탭을 누릅니다.
  • 스프레드시트 아이콘(초록색 십자가)을 클릭하고 '기존 스프레드시트 선택'을 고릅니다.
  • 방금 만든 '확인학습 마스터' 파일을 검색해 연결합니다.

3단계. 앱 배포(URL 만들기) 및 트리거 등록

  • 다시 스프레드시트의 📊 대시보드 ➔ 트리거 등록 메뉴를 클릭합니다. (학생이 폼을 낼 때마다 자동 점수 처리가 되게 하는 엔진입니다.)
  • Apps Script 창 상단 '배포' ➔ '새 배포'를 누릅니다.
  • 유형: 웹 앱 / 실행 주체: 나(me) / 액세스 권한: 모든 사용자(Anyone)로 한 뒤 배포합니다.
  • 발급된 웹 앱 URL (https://script.google.com/macros/s/.../exec)을 복사해 둡니다.

4단계. 대시보드 HTML 연동

  • 컴퓨터에 바탕화면에 있는 dashboard.html 파일을 인터넷 브라우저로 더블클릭해 엽니다.
  • 최초 실행 시 나타나는 "웹 앱 URL 연결" 창에, 3단계에서 복사해둔 웹 앱 URL을 붙여넣고 저장합니다.
  • 연결 성공 메시지가 뜨며 대시보드가 구동되기 시작합니다!

📆 매일의 업무 메뉴얼 (일상 운영)

시트 1: "학생명단" 탭 기록하기

새로 들어온 학생들이 생길 때마다 명단을 채워주세요.

A열 (이름)B열 (학교)C열 (학년)D열 (상태)
김민수유성고1출석 / 지각 / 결석 / 귀가
  • 상태(D열): 매일 등원 시 기본적으로 "출석"이나 "지각"으로 둡니다.
  • 이 상태는 HTML 대시보드 화면상에서도 학생 카드의 버튼으로 언제든지 쉽게 클릭 한 번으로 바꿀 수 있습니다!

시트 2: "할일배정" 탭에 퀴즈 나눠주기

오늘 어떤 학년/학교가 어떤 시험을 치를지 적어둡니다.

A (학교)C (퀴즈제목)D (폼링크)F (대상학생)
유성고 (1학년)독해-모의3차https://forms...비움 (전체)
비움재시-영단어https://forms...이하은, 박지호
💡 팁: 퀴즈 제목(C열)은 구글 폼 제목과 똑같아야 합니다!
특정 학생(F열)에게만 콤마(,) 단위로 분리해서 퀴즈를 지정해줄 수도 있습니다. (예: 보충학습용)

🛠 자주 묻는 질문 & 트러블슈팅

Q. 학생이 폼을 제출했는데 대시보드에 안 떠요!

  • 상단에 노란색 "이름 미매칭 경고"가 뜬 게 아닌지 확인하세요. 학생이 이름을 오타 냈다면(예: '김민수' 대신 '김민슈') 시스템이 짝지어 주지 못해 오류함에 보관됩니다. (Summary 시트에서 오타 이름을 직접 수정해주세요.)

Q. 시트에 학생을 추가했는데 대시보드에 안 보입니다.

  • 과부하를 막기 위해 데이터는 5분 정도 기억(캐싱)됩니다. 바로 보시려면 마스터 시트에서 메뉴 📊 대시보드 ➔ 캐시 초기화를 누르고 대시보드를 새로고침(F5) 해주세요.

Q. 학생이 시험을 망쳐서 재시험을 봐야 해요.

  • 그냥 구글 폼 링크를 주시고 다시 제출하라고 하시면 됩니다. 시스템이 똑똑하게 가장 최신에 제출한 최고차수 점수로 대시보드 점수를 덮어씌워 줍니다! ('2차 90%' 식으로 표기됩니다.)
